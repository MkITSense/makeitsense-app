# App Wifi

Aplicación de ejemplo para el uso de la librería `MKS_Wifi`. Esta aplicación:

- Se conecta a un accespoint
- Utiliza la librería `WifiManager` que permite configurar desde un smartphone


## Materiales

- Sparkfun ESP8266
- Display OLED 0.96 pulgadas
